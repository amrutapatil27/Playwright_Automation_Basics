To write a good locator, you must understand the "bones" of the table.

<table>: The container.

<thead>: The header section (labels).

<tbody>: The body section (the actual data).

<tr>: Table Row. This is your primary target for scoping.

<td>: Table Data (Cell). This contains the actual values.

 We only use the hash symbol (#) when we are looking for a specific id attribute. We do not use # for standard HTML tag names.
